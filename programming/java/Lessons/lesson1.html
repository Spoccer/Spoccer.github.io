Introduction -

I hope that you are excited about getting started with programming in Java! We are going to hit the ground running. You are going to write your first (simple) Java program in this first lesson. It doesn't matter whether you are an experienced programmer or a complete novice, the instructions in this lesson will quickly get you to the goal of writing your first program.

Whenever you have something you want to make, in order to succeed it is important to have the right tools for the job. If you are going to paint a room, you'll need brushes, rollers, stir sticks, drop cloths... Without those tools you'll have a real mess and fingerpaint on the walls.

As rookie Java programmers you are no different. You need the right tools for the job. When we write programs we will be using an IDE (integrated development environment) called BlueJ. It is specifically designed for students learning Java and provides a visually appealing programming environment and takes care of many of the details involved with testing and running a Java program.

In this lesson you will download BlueJ to your computer (it's free!) and learn how to use it. You'll also write your first programs and ay "Hello" to the programming world!!!

Pre-requisites -

There are no computer knowledge pre-requisites for this lesson, however a good understanding of algebra and logic will be needed to succeed in this course. Whenever you have questions, I encourage you to ask questions of your fellow students and learning leader to help you along the way.

Goals -

  1. Students will be able to download and run BlueJ IDE software on their own computer.

Section 2: Compile and run "Hello World"

This is your objective for this section:

Given the starter code and instructions, use BlueJ to write, compile and run your first "Hello World" program.

The basic building block of java programming is a class. To create a new class, click the New Class button on the left of the IDE.

We will name our new class HelloWorld (no spaces!)

The IDE will now show the icon for a new class titled HelloWorld. Right-click on the icon and select Open Editor (or just double-click on it)

When you open the editor it will be full of example code. We would like to code from scratch today, so you can start by erasing all the code in the window. In the next unit we will learn many of the details about classes, however for this program you will need to just copy the code below. 

// My first program!!!


public class HelloWorld
{
    public static void main(String[] args)
    {
         System.out.println("Hello World!");
    }
}

Once you have entered the code, you can return to the project window and click the Compile button

There are two things that might occur:

  • An error message is created and you are brought back to the editor window. (You will need to make sure you have copied the code exactly - perhaps copy it directly from this page and paste it!)
  • The diagonal striped lines go away and you have successfully compiled your program.

Compiling a program converts it from the words and symbols that programmers understand (and you soon will too) to a level where the computer can use it.

Now you are ready to run your program. Right click on the HelloWorld icon and select void main (String[] args) this will run the main method in your HelloWorld class.

A terminal window will pop-up and display "HelloWorld" just as your program asked it to!

Congratulations! You have written your first program.

Now that you run your first program, let's make it interactive!...

Section 3: Use the terminal window to interact

This is your objective for this section:

Make an 'advanced' version of the HelloWorld class that interacts with the user.

Let's add a few things to make the code more interactive. Again, remember that you are not expected to understand any of this code - we're just getting used to using BlueJ. However, you should be able to understand some of the concepts. 

// My first program!!!
import java.util.Scanner;

public class HelloWorld
{
    public static void main(String[] args)
    {
       //Creates a Scanner object connected to the system input (keyboard)
       Scanner keyboard = new Scanner(System.in);
   
       //Display a prompt to the user.
       System.out.println("What is your name?");
   
       //Get the username from the keyboard.
       String userName = keyboard.next();
   
       //Display a personalized message.
       System.out.println("Hello, "+userName);
    }
}

When you run this program, it should ask the user their name, wait for them to type it, and then display a personalized message. It is easy to follow the code since it is full of comments. Comments are put into code to make it easier to read, but are ignored by the compiler. They are just notes to programmers. The comments in this program all start with two slashes ( // ).
